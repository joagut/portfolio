# Portfolio deploy package

Estructura lista para deploy:

- index.html: HTML principal
- css/styles.css: estilos del sitio
- components/translations.js: textos ES/EN
- components/app.js: interacciones, menu, tabs y animaciones
- assets/: imagenes, videos y documentos locales copiados desde la carpeta original

Para deployar, usar esta carpeta como raiz del sitio. `index.html` debe quedar al mismo nivel que `css/`, `components/` y `assets/`.
